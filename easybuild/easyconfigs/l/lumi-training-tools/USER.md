# lumi-training-tools user documentation

The lumi-training-tools is a module-under-continuous-development that provides tools 
used during the trainings, mostly for exercises, but that don't make sense to include
in modules that are useful outside trainings also such as
[lumi-CPEtools](../lumi-CPEtools).

Content of the newest versions:

-   `mkfile`: A simple tool to generate a file of given size.

-   `lstopo`: A command that shows the topology of a cluster node.
    The command is used in the notes of the bindings lecture of the
    intro courses. This version here has slightly more features built-in
    than the version in the `systools` module of 25.03 and later.
